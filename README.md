# 🛍️ Webbutik – Frontend Projekt (G-nivå)  
  
Detta projekt är en webbapplikation som utvecklats som en del av kursen Frontend.    
Applikationen hämtar produkter från Fake Store API och visar dem dynamiskt i en responsiv layout.  
  
Användaren kan se information om varje produkt, såsom titel, pris, bild och beskrivning.    
Det finns även ett beställningsformulär där användaren kan ange sina uppgifter för att beställa en produkt.  
  
Fokus i projektet har varit att arbeta med API-anrop, DOM-manipulation, responsiv design samt validering av formulär med JavaScript.  
  
---  
  
## 🚀 Funktioner  
  
- Visa produkter från API    
- Responsiv design    
- Beställningsformulär    
- Formulärvalidering
